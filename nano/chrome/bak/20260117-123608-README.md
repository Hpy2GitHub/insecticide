# Privacy-First Gemini Nano Chrome Extension

A Chrome extension that runs Google's Gemini Nano model locally on your device using the window.ai API.

## Features
- 100% local inference (no data sent to servers)
- Text generation via Gemini Nano (3-4B parameter model)
- Context menu integration
- Floating assistant button
- Side panel interface

## Requirements
- Chrome 122+ with window.ai API enabled
- Modern GPU (4GB+ VRAM) OR 16GB+ RAM for CPU inference
- Chrome flags enabled: `#optimization-guide-on-device-model`

## Installation

1. Enable Chrome flags:
   - Go to `chrome://flags`
   - Search for and enable:
     - `#optimization-guide-on-device-model`
     - `#prompt-api-for-gemini-nano`

2. Load the extension:
   - Open `chrome://extensions`
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the extension folder

## Usage

1. Click extension icon for quick access
2. Select text on any page → click floating ✨ button
3. Right-click selected text → "Ask Gemini Nano"
4. Use side panel for extended conversations

## Privacy Guarantee
- All processing happens locally
- No internet connection required
- No data logging or tracking
- Your data never leaves your device
